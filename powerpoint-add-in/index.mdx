## In this section

- Users can install Grunt, activate a license, link to Excel, and export or send presentations.
- IT can choose between per-user and per-machine installs, allow required endpoints, provision licenses, and control update behavior.
- Troubleshooting pages cover missing tabs, expired licenses, Excel path issues, Office add-in blocking, and endpoint protection conflicts.
