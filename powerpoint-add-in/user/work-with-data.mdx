Grunt lets you build and update chart objects directly in your slides.

## Create a visual from Excel data

1. In Excel, copy your data table.
2. In PowerPoint, open the **Grunt** tab.
3. Use the Grunt insertion flow and paste the data.
4. Choose the visual type and apply formatting.

<Info>
Use data that already has clear headers and consistent number formatting for faster setup.
</Info>

## Keep data linked

When data is linked, you can refresh visuals as source values change.

- Use linking when you expect frequent updates.
- Use one-time paste when the data is final.

## Useful shortcuts

- `CTRL + ALT + C`: Open chart editor
- `CTRL + ALT + T`: Open table editor
- `CTRL + ALT + G`: Open gantt editor
- `CTRL + ALT + U`: Update linked data
