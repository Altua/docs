title: "Smart Selection"
description: "Use Smart Selection to tidy, space, and resize multiple objects faster"
---

Smart Selection enhances native PowerPoint multi-select behavior with layout controls directly on the slide.

## Turn Smart Selection on

Enable **Smart Selection** from the **Grunt** ribbon before using the features on this page.

When it is active:

- Pink markers show the current reference object
- Pink handles appear between or around selected objects
- Tidy-up previews appear when you hover supported controls

## Use Tidy Up

Use **Tidy Up** when objects should line up as a cleaner group.

The legacy article notes that:

- Grid-like selections are adjusted in both horizontal and vertical directions
- Depending on the start position, spacing can become equal or remain intentionally distinct

Hover the Tidy Up button to preview the result before applying it.

## Use Smart Spacing

Smart Spacing lets you drag the distance between objects instead of calculating it manually.

1. Select the objects.
2. Drag the pink handle that appears between them.
3. Release the mouse when the spacing looks right.

Tips from the legacy support article:

- Hover a handle to see the spacing in pixels
- Hold `Ctrl` while dragging to space objects symmetrically from the center

## Use Smart Resizing

Smart Resizing keeps spacing relationships while you resize multiple objects together.

1. Select the objects.
2. Drag one side or corner.
3. Release to apply the resize.

Modifier keys:

- Hold `Ctrl` to resize symmetrically from the center
- Hold `Shift` to preserve aspect ratio

## Temporarily turn it off

If Smart Selection gets in the way for a specific action:

- Press `Esc` to disable it for the current selection only
- Toggle the ribbon control off to disable it globally

For the wider set of workflow tools, continue to [Tools and shortcuts](/powerpoint-add-in/user/tools-and-shortcuts).
