---

Use **Export** when you need a final deliverable. Use **Send** when you want to email from PowerPoint.

## Export options

Grunt export supports:

- `.pptx` with Grunt objects converted for broader compatibility
- `.pptx` that keeps Grunt objects for future editing
- `.pdf`

You can export:

- Entire presentation
- Current section
- Selected slides
- Current slide

## When to use each format

| Goal | Recommended option |
| --- | --- |
| Share with users who do not have Grunt | `.pptx` with converted objects |
| Keep full Grunt editability | `.pptx` with Grunt objects |
| Share final, fixed output | `.pdf` |

## Send by email

Use **Send** to open a new Outlook email with your generated file attached.

If your organization blocks desktop Outlook integration, export first and attach the file manually.
