Use this guide to install Grunt and verify it is ready in PowerPoint.

## Prerequisites

- PowerPoint for Microsoft 365 on Windows
- Office 2024 for Mac (beta)
- Windows 10/11 or macOS 15+
- A Grunt account and active license

## Install Grunt

1. Download the installer from [app.grunt.pro/download](https://app.grunt.pro/download).
2. Run the installer.
3. Open PowerPoint and confirm the **Grunt** tab appears in the ribbon.

## Activate your license

When you open Grunt for the first time, you should get a license prompt.

If you do not:

1. Open PowerPoint.
2. Go to **Grunt** > **Settings** > **License**.
3. Sign in and complete activation.
