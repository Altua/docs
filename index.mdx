## What to expect

These guides focus on production integrations.

1. You install `@grunt/ckeditor5-grunt` with npm.
2. You register the Grunt plugin in CKEditor 5.
3. You connect a custom storage pipeline to your backend API.
4. You serve WebAssembly runtime assets correctly in production.
