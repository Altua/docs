If your team uses a standard template, set it as the default PowerPoint theme once and let PowerPoint apply it automatically in new presentations.

## Set the default theme

The legacy support workflow is:

1. Open the company theme or template in PowerPoint.
2. Open the **Design** tab.
3. Expand the theme gallery and choose **Save Current Theme**.
4. Give the theme a recognizable name.
5. Open the **Design** tab again and find the saved theme under **Custom**.
6. Right-click the saved theme.
7. Choose **Set as Default Theme**.

PowerPoint now uses that theme automatically for new presentations.

## Why this matters for Grunt

Grunt works inside PowerPoint. If your PowerPoint defaults are correct:

- New decks start in the right company visual style
- Theme fonts and colors are more consistent from slide one
- You spend less time correcting presentation setup before using Grunt

## Font caveat

If a theme relies on embedded fonts, Grunt may still not be able to access those fonts through the Office API. In that case, install the font locally or switch to an available font. See [FAQ and tips](/powerpoint-add-in/user/faq-and-tips).
